# simple-image-interface

- This repository is simple image interface library for rust
- Support image interface
  - [ x ] Camera input (v4l2)
  - [ ] Realsense
  - [ ] Basler Camera
  - [ ] video (mp4)
  - [ x ] image (png, jpg)
